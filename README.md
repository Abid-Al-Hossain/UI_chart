# UI Foundry Chart Studio

Standalone, self-contained Chart component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Source model: src, poster/thumbnail, captions/tracks, alt/title, fallback states
- Presentation: aspect ratio, fit, object position, overlays, controls, frame and mask
- Interaction: playback or slide controls where native, keyboard and gesture-safe previews
- Accessibility: captions, labels, decorative mode, autoplay/motion safety
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
